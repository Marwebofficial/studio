# **App Name**: Web Chat Navigator

## Core Features:

- Question Input: A text input field where users can type their questions.
- Web Search: Search the web using the question as a query and fetch relevant information.
- AI Answer Generation: Use a large language model to generate a comprehensive answer based on the search results. The LLM uses a tool to decide when and whether information should be added.
- Answer Display: Display the generated answer to the user in a clear and readable format.
- Source Links: Provide links to the original web pages used to generate the answer.

## Style Guidelines:

- Primary color: A calming blue (#64B5F6), suggestive of knowledge and reliability.
- Background color: Light gray (#F0F4F8), for a clean and focused interface.
- Accent color: A vibrant orange (#FF9800) to highlight key elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable text.
- Use simple, consistent icons for sources and navigation.
- Clean, minimal layout with a focus on readability and easy navigation.
- Subtle animations for loading and transitioning between answers.
